# 会理石榴产品宣传网站设计风格

## 设计理念

### 颜色方案
- **主色调**: 温暖的石榴红色 (#C0392B) 和深绿色 (#27AE60)
- **辅助色**: 米白色 (#F8F9FA) 和浅灰色 (#ECF0F1)
- **强调色**: 金黄色 (#F39C12) 用于突出重要元素
- 整体色彩搭配体现石榴的自然色彩和品牌识别度

### 排版设计
- **标题字体**: 使用现代无衬线字体 "Playfair Display" 营造优雅感
- **正文字体**: 使用 "Inter" 字体确保可读性和现代感
- **字体层次**: 大标题 (48px+) 配合小正文 (16px) 创造视觉层次
- 文字颜色主要使用深灰色 (#2C3E50) 确保对比度

### 视觉语言
- 自然有机的设计元素，体现农产品的天然属性
- 简洁现代的布局，突出产品本身的美感
- 使用大量留白空间，营造高端品质感
- 圆角设计和柔和阴影，增加亲和力

## 视觉效果

### 使用的库和技术
- **Anime.js**: 实现流畅的动画过渡效果
- **ECharts.js**: 创建数据可视化图表
- **Splide.js**: 产品图片轮播展示
- **p5.js**: 创意背景粒子效果
- **Pixi.js**: 高性能的视觉特效

### 动画效果
- 页面滚动时的渐入动画
- 产品卡片的悬停3D倾斜效果
- 导航菜单的平滑过渡
- 按钮点击的涟漪效果
- 背景粒子缓慢移动营造动态感

### 样式特色
- 卡片式设计，每个产品独立展示
- 渐变背景从浅红到浅绿的柔和过渡
- 产品图片使用圆角和阴影增强立体感
- 导航栏半透明设计，保持内容可见性
- 响应式网格布局适配各种屏幕尺寸

### 头部效果
- 使用 p5.js 创建石榴主题的背景动画
- 粒子系统模拟石榴籽的动态效果
- 渐变色背景配合动态光效
- 标题文字使用颜色循环强调效果
