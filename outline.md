# 会理石榴产品宣传网站项目大纲

## 网站结构

### 1. 主页 (index.html)
**功能定位**: 品牌展示和产品亮点
**主要内容**:
- 导航栏：品牌Logo + 主要页面链接
- 英雄区域：大型石榴背景图 + 品牌标语 + 主要行动按钮
- 产品轮播展示：精选石榴产品自动轮播
- 品质特色介绍：会理石榴的三大优势
- 客户评价区域：展示用户反馈
- 底部联系信息

**交互功能**:
- 产品轮播器（自动+手动控制）
- 滚动动画效果
- 悬停产品卡片显示详情

### 2. 产品展示页 (products.html)
**功能定位**: 完整产品目录和购买功能
**主要内容**:
- 产品筛选侧边栏：按品种、大小、价格筛选
- 产品网格展示：每个产品包含图片、名称、价格、简介
- 产品详情模态框：点击产品显示详细信息
- 购物车功能：添加商品、数量调整、价格计算

**交互功能**:
- 多条件产品筛选器
- 购物车实时更新
- 产品图片放大查看
- 数量选择和规格选择

### 3. 关于会理页 (about.html)
**功能定位**: 产地介绍和品牌故事
**主要内容**:
- 会理地理介绍：气候、土壤、环境优势
- 石榴种植历史：传统工艺和现代技术
- 产地地图展示：交互式地图显示种植区域
- 品质认证展示：各种质量认证和奖项
- 农民故事：当地种植户的介绍

**交互功能**:
- 交互式地图（点击区域显示信息）
- 时间线展示种植历程
- 图片画廊展示种植过程

### 4. 联系购买页 (contact.html)
**功能定位**: 购买咨询和联系方式
**主要内容**:
- 联系表单：姓名、电话、需求描述
- 联系方式：电话、微信、邮箱
- 公司地址和地图位置
- 批发咨询信息
- 配送服务说明

**交互功能**:
- 联系表单验证和提交
- 地图定位显示
- 在线客服入口

## 技术实现

### 文件结构
```
/mnt/okcomputer/output/
├── index.html          # 主页
├── products.html       # 产品页
├── about.html          # 关于页
├── contact.html        # 联系页
├── main.js            # 主要JavaScript功能
├── resources/         # 资源文件夹
│   ├── hero-bg.jpg    # 英雄区背景图
│   ├── product-*.jpg  # 产品图片
│   └── about-*.jpg    # 关于页面图片
└── README.md          # 项目说明
```

### 核心功能模块
1. **导航管理**: 页面间跳转和状态保持
2. **产品数据管理**: 产品信息存储和展示
3. **购物车逻辑**: 商品添加、删除、数量调整
4. **筛选器功能**: 多条件产品筛选
5. **动画效果**: 页面切换和元素动画
6. **表单处理**: 联系表单验证和提交

### 响应式设计
- 桌面端：1200px+ 完整功能展示
- 平板端：768px-1199px 适配触摸操作
- 移动端：<768px 简化界面和手势支持
